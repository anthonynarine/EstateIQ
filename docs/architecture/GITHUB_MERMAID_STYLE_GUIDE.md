# GitHub Mermaid Style Guide

Use Mermaid features that render reliably on GitHub:

- `flowchart`
- `sequenceDiagram`
- `erDiagram`

## Practical rules

- one main idea per diagram
- prefer left-to-right or top-to-bottom flow
- keep labels short
- explain the diagram directly below it
- do not hide domain ownership
- do not make diagrams decorative only
