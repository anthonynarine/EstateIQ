# Expenses Domain

The expenses domain is one of the system’s core financial truth sources.

## Owns

- expense records
- categories
- vendors
- attachments / receipts
- expense-specific reporting selectors and contracts

## Does not own

- occupancy truth
- lease lifecycle
- rent charges and payments
- universal cross-domain reporting logic
