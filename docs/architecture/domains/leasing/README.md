# Leasing Domain

The leasing domain owns the contract layer between a tenant and a unit.

## Owns

- tenants
- leases
- parties on a lease
- occupancy truth

## Does not own

- monthly rent posting mechanics
- payment recording
- portfolio-wide expense reporting
