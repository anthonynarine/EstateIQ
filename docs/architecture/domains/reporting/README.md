# Reporting Domain

Reporting turns deterministic facts from other domains into readable portfolio insight.

## Owns

- aggregate selectors
- dashboard payload composition
- chart contracts
- report exports

## Does not own

- source-of-truth CRUD for billing or expenses
- lease lifecycle logic
- mutation-heavy business rules
