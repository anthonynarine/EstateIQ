# Billing Domain

The billing domain is the **receivables subledger** for a lease.

## Owns

- charges
- payments
- allocations
- lease ledger
- delinquency logic
- internal billing alerting

## Does not own

- tenant portal payments in MVP
- automatic assumptions about rent posting
- portfolio-wide dashboard composition
