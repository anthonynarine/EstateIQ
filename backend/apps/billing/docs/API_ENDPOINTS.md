# API Endpoints

## Lease Ledger
GET /api/v1/leases/{lease_id}/ledger/

## Generate Specific Month
POST /api/v1/leases/{lease_id}/charges/generate-month/

## Generate Current Month
POST /api/v1/leases/{lease_id}/charges/generate-current-month/

## Create Payment
POST /api/v1/payments/

## Delinquency Report
GET /api/v1/reports/delinquency/

## Dashboard Summary
GET /api/v1/reports/dashboard-summary/

## Bulk Rent Posting
POST /api/v1/reports/rent-posting/run-current-month/
